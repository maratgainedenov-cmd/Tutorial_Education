# Типы блоков

## Таблица

| Тип | PieceType | IsWeak | HP | Описание |
|-----|-----------|--------|----|----------|
| Сильный | 1–7 | false | 0 | Обычный тетромино, персонаж не может разбить |
| Слабый | 8+ | true | N | Персонаж может разбить ударом, имеет HP |

## CellData (struct)

```csharp
public struct CellData
{
    public int  PieceType;  // 0 = пусто, 1-7 = сильный, 8+ = слабый
    public bool IsWeak;
    public int  HP;

    public bool IsEmpty => PieceType == 0;
}
```

## Спавн слабых блоков

- Вероятность слабой фигуры: `_weakPieceChance` (по умолчанию 30%)
- HP слабого блока: `_weakBlockHP` (по умолчанию 2)
- Вся фигура целиком либо сильная, либо слабая (не по-клеточно)

## Взаимодействие с персонажем

- Сильный блок: персонаж **не может разбить**, просто упирается
- Слабый блок: `DamageCell(x, y)` → HP--; если HP ≤ 0 → ячейка очищается → очко персонажу
