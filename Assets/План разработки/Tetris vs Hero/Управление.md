# Управление

## Персонаж

| Действие | Клавиша |
|----------|---------|
| Движение влево | A |
| Движение вправо | D |
| Прыжок | W |
| Удар влево | J |
| Удар вправо | L |

> Каждый удар снимает 1 HP с блока. При HP = 0 блок разрушен (+1 очко).
> Подробнее → [[Типы блоков]]

## Тетрис-игрок

| Действие | Клавиша |
|----------|---------|
| Движение ← | ← |
| Движение → | → |
| Поворот по часовой | ↑ |
| Поворот против часовой | Z |
| Soft drop | ↓ |
| Hard drop | Space |

## Настройка

Все клавиши задаются через `[SerializeField] KeyCode` в Inspector — менять без перекомпиляции.
